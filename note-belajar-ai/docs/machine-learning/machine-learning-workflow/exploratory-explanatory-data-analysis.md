# Exploratory dan Explanatory Data Analysis

![EDA](image-2.png)

Exploratory Data Analysis (EDA) merupakan tahap eksplorasi data yang telah dibersihkan guna memperoleh insight dan menjawab pertanyaan analysis. 
Explanatory Data Analysis (ExDA) merupakan tahap dengan tujuan untuk mengkomunikasikan data temuan atau insight yang sudah didapat kepada audiens yang lebih luas.
